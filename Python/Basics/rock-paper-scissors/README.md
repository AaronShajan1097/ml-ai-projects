# Rock Paper Scissors

## Description
A command-line Rock Paper Scissors game where the user plays against the computer.
The computer randomly selects a move and the winner is decided based on game rules.

## Concepts Used
- Python lists
- Random module
- Conditional statements
- User input handling

## Learnings
- Using lists and indexing
- Generating random choices
- Implementing game logic

## How to Run
python rock_paper_scissors.py

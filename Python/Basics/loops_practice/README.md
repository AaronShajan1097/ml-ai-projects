# Loop Practice Programs

## Description
A collection of beginner-level Python programs designed to practice loops
and basic problem-solving techniques.

## Programs Included
- Adding Even Numbers
- Average Height
- High Score
- Password Generator
- Sum of Numbers
- FizzBuzz

## Concepts Used
- for loops
- while loops
- Conditional logic
- Accumulators and counters

## Learnings
- Iterative problem solving
- Writing efficient loops

## How to Run
python adding_even_numbers.py
python average_height.py
python high_score.py
python password_generator.py
python sum_of_numbers.py
python fizz_buzz.py

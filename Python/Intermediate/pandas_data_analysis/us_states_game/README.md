# U.S. States Guessing Game

## Description
An interactive game where users guess U.S. states and learn geography
using Python and Turtle graphics and Correct guesses are displayed on a map.

## Concepts Used
- Pandas
- File handling
- Turtle graphics
- User input handling

## Learnings
- Combining data with visual interaction
- Tracking progress using files

## How to Run
python main.py
